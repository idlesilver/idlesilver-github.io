                                            # 段落

                                            Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，然后转换成格式丰富的HTML页面。 —— 维基百科open in new window

                                            使用简单的符号标识不同的标题，将某些文字标记为粗体或者斜体，创建一个 链接open in new window 等。

                                            本编辑器支持 Markdown Extra , 　扩展了很多好用的功能。具体请参考.
